# 反 AI 体检清单（Deck Anti-Slop Checklist）

出货前**强制**逐条过。命中任何一条，回去改。
这不是建议，是交付门槛。

---

## 一、致命烂 deck 特征（命中=必改）

一眼就能看出「AI 糊的字墙 PPT」，出现一个就回去重做那部分：

- [ ] **标题 + 三个等长 bullet**：整套 deck 每页都这个结构。
- [ ] **母版套壳**：每页一样的版式、配色、节奏，从头平铺到尾，没有封面 / 过渡 / 收尾的区分。
- [ ] **一页多个观点**：一页塞了两三件事，没有单页焦点。
- [ ] **通篇字墙**：满页都是字，没有一个视觉锤，没有一页能被记住。
- [ ] **正文当文档写**：每个 bullet 两三行完整句子，观众在读不在听。
- [ ] **emoji 当图标**撑场面；用「谢谢聆听 / Thank you」四个字草草收尾。
- [ ] **正确的废话**：「赋能」「一站式」「打造闭环」「持续深耕」这类填充词。

## 二、一页一句话体检

- [ ] **每页能用一句话概括吗？** 抄不出一句话的页，说明没想清楚，回去拆。
- [ ] **一页只有一个主角观点吗？** 还是塞了好几个？
- [ ] **把每页的一句话连起来读，是一条清楚的主线吗？** 还是一堆并列的块？

## 三、黄金三秒体检（投屏视角）

- [ ] **扫一眼 3 秒能 get 这页要说什么吗？**
- [ ] **每页有且只有一个视觉主角吗？** 还是三个东西在抢「我最重要」？
- [ ] **台下最后一排看得清吗？** 字号够大吗？正文别低于 22px，标题敢往 48px+ 上。
- [ ] **对比够狠吗？** 主角和配角的尺寸差，敢不敢再拉大一倍？

## 四、节奏体检

- [ ] **封面立住气质和主线了吗？**
- [ ] **有起伏吗？** 还是每页一个调子、一个版式平铺到尾？
- [ ] **强页之后有喘息吗？** 连续好几页密集数据，观众会睡。
- [ ] **高潮页是不是你最想让人记住的那页，且放在了节奏高点？**
- [ ] **收尾是一个明确行动 / 一句记得住的话吗？** 还是「谢谢聆听」？

## 五、视觉锤体检

- [ ] **整套 deck 有三五个视觉锤吗？**（大数字 / 对比 / 图形 / 宣言）
- [ ] **最关键的那个观点，有没有一个让人记住的视觉？**
- [ ] **能用数字 / 图 / 图形说清的，是不是还在用一段文字？**

## 六、字体与排版

- [ ] **选了非默认字体吗？** 中文带系统兜底（PingFang / YaHei）了吗？
- [ ] **中文没用日文字体渲染吧？**（会缺字 / 异体字）
- [ ] **大标题折行没有把单字 / 标点甩到独立一行吧？**（孤字）
- [ ] **没用 ch 给中文限宽吧？**（中文会算错宽度溢出）

## 七、工程质量（投屏 / 导出 / 性能）

- [ ] **键盘能翻页吗？**（← → 空格）有页码 / 进度吗？
- [ ] **16:9 铺满不变形吗？** 不同屏幕缩放正常吗？
- [ ] **能 Cmd+P 导出每页一张的 PDF 吗？**
- [ ] **不卡吗？** 没动画化阴影、没 backdrop-filter、没动画化模糊元素吧？
- [ ] **Google Fonts 挂了中文还正常吗？**（有系统兜底）

---

## 终极一问

> **"如果我现在就要拿这套 deck 上台，敢吗？讲完台下有人能复述出我的主线、记得住我那几页吗？"**

答"敢"且"能"，才能交付。
