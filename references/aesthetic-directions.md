# 气质方向库（Deck Aesthetic Directions）

定调时选一个**极端**方向，或在此基础上自创。
原则：宁可走极端走出记忆点，也不要选「商务简约」这种安全的废话。

每种方向给出可直接落地的打法：色彩、字体、封面、视觉锤怎么做。

---

## 1. 凶悍 / 工业（Brutalist）

气质：硬核、高对比、有攻击性、敢下判断。适合创业路演、硬科技发布、有态度的方案。

- **色彩**：近黑底 + 一个高饱和电光色（电光橙 / 酸绿 / 警示黄）。绝不用柔和渐变。
- **字体**：超粗无衬线（Archivo Black、Anton、思源黑体 Heavy）+ 等宽体做标注。
- **封面**：超大字宣言占满屏，刻意不对称，裸露网格线。
- **视觉锤**：巨型数字 + 单位、硬边对比块、占满屏的一句狠话。
- **页码 / HUD**：等宽体、像仪表盘。

## 2. 优雅 / 高定（Editorial）

气质：精致、有文化气、克制的高级感。适合方案提案、品牌发布、咨询汇报。

- **色彩**：纸感米白 / 暖灰 + 一个深色（墨蓝 / 酒红）+ 至多一个点缀金。低饱和。
- **字体**：高质量衬线（Playfair Display、思源宋体）做标题 + 干净无衬线正文。
- **封面**：大量留白 + 一行衬线大字 + 一条细分割线，杂志扉页感。
- **视觉锤**：一句超大号衬线引言、一组精排的数据、一张满版出血图位。
- **过渡页**：纯色 + 一个章节号，像翻章节。

## 3. 克制 / 极简（Swiss / Data）

气质：纪律、秩序、信息清晰、专业可信。适合年度复盘、财报、数据汇报、B2B。

- **色彩**：黑白灰 + 一个功能强调色（信号红 / 蓝）。色彩极少，靠排版说话。
- **字体**：Inter / Helvetica / 思源黑体，严格字号阶梯。
- **封面**：左对齐标题 + 大量留白 + 一个关键数字。
- **视觉锤**：极致干净的图表、一个被强调色点亮的关键数、对齐到像素的表格。
- **节奏**：靠留白和尺寸分层级，不靠花哨。

## 4. 科技 / 未来（Tech / Neon）

气质：前沿、酷、产品感。适合 AI / 软件产品发布、技术 keynote、demo day。

- **色彩**：深空底（近黑 / 深蓝）+ 霓虹强调（青 / 品红 / 电蓝），克制使用，大面积留暗。
- **字体**：科技感无衬线（Space Grotesk、Chakra Petch）+ 等宽体（JetBrains Mono）做代码 / 标注 + 思源黑体。
- **封面**：产品名超大 + 一句 tagline + 一个发光的几何 / 网格元素。
- **视觉锤**：发光的关键指标、一段终端 / 代码、一条上扬曲线、产品截图位的强调框。
- **注意**：霓虹只做点缀和描边，别整屏发光晃眼；遵守性能规格（别动画化阴影）。

## 5. 张扬 / 极繁（Maximalist）

气质：热闹、年轻、有冲击。适合技术分享、活动招商、面向年轻人的发布。

- **色彩**：撞色、高饱和，渐变可以大胆用（别用紫蓝那套烂大街的）。
- **字体**：多字体混搭、超大字、变形字。
- **封面**：标题占满屏 + 贴纸 / 几何元素堆叠，先声夺人。
- **视觉锤**：一面信息墙、夸张的大字金句、强对比的前后图。
- **节奏**：开场要炸，但中间论证段要适当收，否则观众疲劳。

## 6. 温暖 / 人文（Warm / Humanist）

气质：亲切、有温度、讲故事。适合培训课件、公益、教育、面向大众的分享。

- **色彩**：大地色 / 暖色低饱和 + 柔和过渡，纸张 / 材质感。
- **字体**：圆润无衬线 / 温暖衬线（霞鹜文楷 LXGW WenKai 很合适）+ 手写体点缀。
- **封面**：一句走心的话 + 一个手绘 / 有机形状元素。
- **视觉锤**：一张大的情绪图位、一个故事性的对比、一句温暖的金句。

---

## 跨方向通用的"反平庸"硬规则

无论选哪个方向，这几条都适用：

1. **一页一个主角**。字号 / 尺寸对比敢拉大，主角大到「有点过分」才够投屏。
2. **强制选一个非默认字体**，中文带系统兜底（见 `deck-production-spec.md`）。
3. **色彩克制**：主色 + 至多两个辅助色 + 中性色。节奏靠版式，不靠堆色。
4. **每套 deck 至少三五个视觉锤**，最关键的观点必须有一个。
5. **封面、过渡页、收尾页各有专门版式**，不要全套一个母版到底。
