# 节奏曲线模板（Deck Pacing Templates）

整套 deck 是一场演出，不同场景有不同的节奏排布。
下面是常见类型的页面节奏参考。**模板是起点，不是终点**，拿到具体内容要按主线重新导演。

记号：⚡ 抓人 / 🩹 痛点 / ↪ 转折 / 📈 论证 / 🔨 视觉锤页 / 🫁 喘息 / 🎬 收束

---

## 创业路演 Pitch Deck（给投资人，要他投钱）

```
⚡ 封面          一句话说清你是谁、做什么，立气场
🩹 痛点          一个真实、够痛的问题，最好配一个数字
↪ 解法          你怎么解，一句话 + 一张图
🔨 这有多大      市场规模 / 增长，巨型数字砸下来
📈 凭什么是你    产品 / 技术 / 壁垒，一页一个点
🔨 已经跑通了    关键牵引数据（用户 / 收入 / 增长曲线）
📈 怎么赚钱      商业模式，一句话
🫁 团队          为什么这帮人能干成
🎬 要钱 + 用途   融多少、干什么、达成什么，干脆利落
```
要点：投资人只给 10 分钟，主线必须是「痛点够大 → 解法够巧 → 数据够硬 → 团队够强 → 给钱」。牵引数据页是全场高潮。

## 产品发布 Keynote（给用户 / 媒体，要他们记住并想用）

```
⚡⚡ 封面         产品名超大 + 一句 tagline，悬念感
🩹 旧世界        现在有多麻烦 / 多别扭，铺垫
↪ One more thing 产品登场，一句话定义它是什么
🔨 核心能力①     一个能力一页，配 demo 截图位 / 大字
🔨 核心能力②     同上，节奏推进
🔨 杀手锏        最炸的那个功能，放在高潮位，给视觉锤
🫁 一句金句      留白页，让那句 slogan 占满屏
📈 价格 / 上市    什么时候、多少钱
🎬 行动          去哪下载 / 预约，一个动作
```
要点：学发布会的悬念节奏。先放大旧世界的痛，再让产品「登场」。杀手锏页是全场记忆点。

## 年度复盘 Annual Review（给老板 / 团队，要他们看清并认可）

```
⚡ 封面          一年一句话定调（如「这一年，我们把 X 跑通了」）
🔨 一年成绩      最关键的那个数字，巨大，单独一页
📈 三件大事      做成的三件事，一件一页，每件配数据
🩹 踩过的坑      诚实复盘，没做好的，一页（加信任分）
📈 怎么改的      针对坑的动作
📈 数据全景      关键指标看板，一页给密度
🫁 一句感谢 / 总结 留白
🎬 明年押什么    下一年的方向和目标，一个清晰的主攻点
```
要点：成绩页要敢用巨型数字。坦诚讲坑反而加分。收尾给明年的方向，不要停在「谢谢」。

## 方案提案 Consulting Proposal（给客户，要他签单）

```
⚡ 封面          提案标题 + 给谁，正式立信任
🩹 我们理解的问题 先复述客户的痛，证明你懂他（关键）
↪ 我们的判断     你对问题的独到判断，一句话
📈 方案框架      整体怎么做，一张结构图
🔨 关键打法      最有亮点的一两个动作，给视觉锤
📈 落地路径      分几步、各做什么、时间线
🔨 预期效果      能带来什么，量化，给数字
🫁 我们凭什么    团队 / 案例 / 资质
🎬 下一步        合作方式 + 一个明确的推进动作
```
要点：开场先证明你懂客户的痛，比一上来吹自己有用得多。效果页要敢量化。

## 技术分享 / 演讲 Talk（给同行 / 学员，要他们学到并记住）

```
⚡ 封面          话题 + 一个钩子问题，勾住好奇
🩹 为什么讲这个   一个共鸣点 / 一个反常识
📈 核心观点①     一个观点一页，配例子
🔨 关键洞察      最有价值的那个点，给视觉锤
🫁 一个故事      讲个真实的小故事，换口气
📈 核心观点②③    继续推进
🔨 一张图总结     把整套观点收成一张图 / 一个框架
🎬 三句话带走    最想让人记住的三点 + 一个行动建议
```
要点：开场用问题或反常识勾人。中间穿插故事换节奏。收尾给「带走的三句话」，别让人空手走。

## 通用导演口诀

1. **封面定生死**：第一页立不住气场和主线，后面全靠观众硬撑。
2. **一页一句话**：讲不清的拆页，别挤。
3. **强弱交替**：密集论证页之后给一页留白 / 金句，让观众喘气。
4. **高潮留给视觉锤**：把最想让人记住的观点放在节奏高点，给它一整页和一个狠视觉。
5. **收尾要行动**：观众走的时候带着一个明确动作或一句话，别用「谢谢聆听」收场。
